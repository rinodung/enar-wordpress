A Beautiful, Professional and Ultimate Wordpress Theme.

Installation

Uploading Enar via FTP

  • 1Log into your hosting space via an FTP software.
  • 2Unzip the Enar.zip file and ONLY use the extracted Enar theme folder.
  • 3Upload the extracted Enar theme folder into wp-content > themes folder.
  • 4Activate the newly installed theme by going to Appearance > Themes and clicking the Activate button.
  • 5Once you activate Enar, You will then see a notification message to install and activate the required & recommended plugins: Revolution Slider, Visual Composer.
  • 6In the notification message, click the "Begin activating plugins" button to go to the "Install Plugins" tab where you can install and activate the required & recommended plugins.
  • 7Simply click the Install button on each plugin to start installing them. After you've installed them, an Activate button will appear which you can click to start activating them.

Installing Enar Using WordPress Dashboard

  • 1Unzip the Enar.zip file and ONLY use the extracted Enar theme folder.
  • 2Log into your WordPress admin panel and click on the "Appearance" tab.
  • 3Now click on the "Add New Themes" tab.
  • 4Click on the "Upload" button, "Browse" for file called Enar.zip In the extracted folder select "Open".
  • 5Click the "Install Now" button.
  • 6Activate the theme from this page, OR "Return to Themes Page" and activate from there.

Updating Enar Theme

This theme may be updated periodically in order to add new features and fix bugs. When a new update be avilable an update message will appears on your blog dashboard . As a customer, you can download the most current version from your ThemeForest account.

Updating Automatically Using Envato Toolkit Plugin

Envato have developed the Envato Toolkit Plugin for WordPress. You can use this to receive notification of updates to themes purchased from ThemeForest and to automatically update (upon clicking) directly from within your WordPress admin area.

  • 1Download the plugin zip file to your computer.
  • 2In your WordPress admin area, go to Plugins > Add New then click Upload.
  • 3Click Choose File, select the plugin zip file you downloaded and click Install Now.
  • 4After installation, click Activate.
  • 5Click the new Envato Toolkit link in the menu and follow the instructions to configure the plugin.

Updating Manually

  • 1Download the most current version from ThemeForest in the "Downloads" area of your account.
  • 2Unzip the package and locate Enar.zip in the theme folder.
  • 3Go to Appearance > Themes and activate another theme such as the default WordPress theme.
  • 4Delete the Enar theme which is now inactive.
  • 5Go to Install Themes > Upload then install and activate Enar.zip from the new package.

Installing Sample Content

Importing demo data (post, pages, images, theme settings, ...) is the easiest way to setup your theme. It will allow you to quickly edit everything instead of creating content from scratch.

  • 1Go to Dashboard > Tools > Import
  • 2Click WordPress link
  • 3Click on Browse and Navigate to folder called "Sample Content", that exist inside theme pack
  • 4select the File Called enar.wordpress.xml and click open
  • 5Click on Upload file and import Button
  • 6Select user that you want this imported content assign to him
  • 7Check Download and import file attachments
  • 8Then Click on Submit button to start import simple data with its images like or demo.

Language Translations

All of our Themes are able to be translated into any language. The process of translating the theme is a user responsibility. The following instructions are given as guidance.

  • 1Open wp-config.php file located directly in your root and replace this: define ('WPLANG', ''); with your own, for example: define ('WPLANG', 'tr_TR');
  • 2Download and install POEDIT
  • 3Connect to your site >> open your theme/lang directory
  • 4open the tr_TR.po file and edit it with POEDIT.
  • 5Translate file and save it as tr_TR ( the file name must match with the string you inserted into wp-config.php )
  • 6Two files will be generated, tr_TR.po and tr_TR.mo
  • 7Upload the .mo and .po files into wp-content/themes/enar/lang folder

Theme Menus

This Theme uses wordpress built-in menus feature which can be found in your wordpress dashboard > appearance > menus. To setup a new menu, please follow the steps below.

  • 1Navigate to Apperance > Menus page
  • 2Click the "create a new menu" link to make a new menu. Enter the name then hit Create Menu.
  • 3To add a menu item, you can select one of your pages on the left side and click Add to Menu.
  • 4You can also add menu items by entering a custom name and custom link into the Links box.
  • 5After setting up your menu, select the menu you just created as the Primary Menu 1 in the Menu Settings and Create a new menu and select it as the Top Menu Navigation in the Menu Settings
  • 6Once its all done, make sure you click the Save button.
  • 7You can chnage menus locations in any time from Manage Locations tab.

Mega Menu

  • 1Navigate to Apperance > Menus page, and Select a menu to edit.
  • 2Go to the menu structure, and click the arrow on the right of the item to reveal additional configuration options.
  • 3Go to "Sub Menu Type" and select "Mega Menu", also you should select the same option for all child items.
  • 4Once its all done, make sure you click the Save button.

Slider Menu

  • 1Navigate to Apperance > Menus page, and Select a menu to edit.
  • 2Go to the menu structure, and click the arrow on the right of the item to reveal additional configuration options.
  • 3Go to "Sub Menu Type" and select "Slider Menu", also you should select the same option for all child items.
  • 4Once its all done, make sure you click the Save button.

Sliders

  • 1Add new slider by navigation to Dashboard > Sliders > Add New.
  • 2Set the slider text title.
  • 3You can use the shortcode text inside pages and posts editor, to show your slider any where.
  • 4Choose the slider type ( Default Slider - PNG Slider - etc... )
  • 5Enter your slide image, title , link and caption.
  • 6Use "Add New Slide" button to add a new slide.
  • 7Set your slider options, Animation Type, Duration speed and etc...
  • 8Once its all done, make sure you click the Publish / Update button.

Page Options

Enar is extremely powerful and a lot of that power comes from our individual page meta options. Each page you create has a full set of options to choose from allowing you to customize each individual page. Each page you open in your WordPress admin will have a Page Options that contains all the different settings. There are several different sections included in the Page Options box, each with its own title and purpose. Some settings will only work if you use a specific template, for example the "For Masonry Colored Template" are only to be used if you select one of our portfolio page templates ( Portfolio Masonry Template ).

Theme Options

Enar has an advanced Theme Options panel that is loaded with options.
